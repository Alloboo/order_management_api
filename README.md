## 3. API 설계

### 구현 API

1. 고객정보 및 주문내역정보 업로드 API: 엑셀 파일을 읽어서 Database에 데이터를 등록한다.
2. 월별 매출 통계 API: 월별 매출 합계 데이터를 반환한다.
3. 주문 목록 조회 API: 주문 목록을 조회하는 페이지에서 API 요청 시 조건에 맞게 주문 데이터를 반환한다.
